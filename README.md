QR Restaurant Management System

A full-stack MERN application where restaurant owners can register, create their restaurant profile, add menu items with images, and generate a unique QR code linked to their restaurant menu. 
Customers can scan the QR code to instantly access the menu, place orders, and track their order status without installing any application.

Technologies Used

React.js,
Node.js, 
Express.js, 
MongoDB Atlas,
Mongoose,
JWT,
bcrypt,
Cloudinary,
Multer,
Axios, 
QRCode,
node-cron
